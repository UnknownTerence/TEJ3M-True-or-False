TEJ3M Summative

What is the Objective?

  The objective of this project is to create a computer device using an arduino and other electronic components. These will need to be programmed using C++ in order for the components to work together to make a device.

  I decided to use push buttons along with an LCD 16x2 to make a small true or false game. There will be multiple questions related to what we have done throughout this computer engineering course.

Future Improvements

  First of all, this project is pretty simple. I decided to do true or false because I was able to get the LCD working. Another reason was because I was planning on using a keypad to make a math game, but I felt that it was too ambitious for the time I had, especially after the python text game I made. What could I have done better? Here's a list for what I think I could've done.

  • Randomized questions
  • More features ie. LEDs and/or buzzers indicating right or wrong
  • The ability to select an amount of questions
